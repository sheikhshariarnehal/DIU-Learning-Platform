## Color Scheme

### Topic Cards
- **Border Left**: Purple (2px) for topics
- **Border Left**: Blue (2px) for study tools
- **Background**: White/Card background
- **Hover**: Subtle accent background

### Drag Handle
- **Normal**: Muted gray
- **Hover**: Accent background, rounded
- **Active**: Cursor changes to grabbing

### During Drag
- **Dragged Item**: 50% opacity, shadow-lg
- **Other Items**: Normal appearance
- **Drop Zones**: Indicated by spacing

## Accessibility Features

1. **Keyboard Support**: 
   - Tab to focus on grip handle
   - Space/Enter to activate drag
   - Arrow keys to move
   - Escape to cancel

2. **Screen Reader**:
   - Announces drag start/end
   - Describes current position
   - Notifies of position changes

3. **Visual Indicators**:
   - Clear cursor changes
   - Hover states
   - Focus outlines

## Responsive Behavior

### Desktop (> 768px)
- Full layout with all features visible
- Smooth drag animations
- Hover effects active

### Tablet/Mobile (< 768px)
- Touch-friendly drag areas (larger hit zones)
- Touch hold to activate drag (8px threshold)
- Simplified layout for narrow screens

## Animation Details

- **Transform**: Smooth position changes during drag
- **Transition**: 150ms ease-in-out for non-dragging items
- **Opacity**: Fades to 50% when dragging
- **Shadow**: Elevates with shadow-lg during drag
